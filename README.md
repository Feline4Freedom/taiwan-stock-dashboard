# 台股全景儀表板 · Taiwan Stock Dashboard

涵蓋 **全部 1,975 家台股上市櫃公司 + 主要 ETF** 的互動式儀表板，依產業分類，顯示公司之間的供應鏈、集團控股、同產業 cluster 與歷史股價相關性。

## ✨ 特色

- **1,975 家公司**（TWSE 1,084 + TPEx 884 + 主要 ETF 7） × **36 個產業** × **172 條連結關係**
- 200+ 家權值股有完整業務描述、產品線、所屬集團
- 其他公司有基本資料（股號、名稱、產業、董事長、官網、成立年份）
- **互動式網路關係圖**（vis-network）：可拖曳、縮放、篩選連結類型
- **產業總覽**：按產業分組瀏覽所有公司
- **公司詳情面板**：基本資料、主營業務、產品、所屬集團、相關公司
- **外部資料連結**：一鍵跳轉 Goodinfo、Yahoo 股市、公開資訊觀測站、鉅亨網、Histock、玩股網
- **深色主題**、繁體中文介面、純前端（無需後端）

## 📊 連結類型

| 類型 | 顏色 | 說明 |
|---|---|---|
| 供應鏈 | 藍色 | 上下游、代工、原料供應關係 |
| 同產業 | 紫色 | TWSE 產業分類同業 |
| 集團/控股 | 橘色 | 集團控股、轉投資、交叉持股 |
| 歷史相關性 | 綠虛線 | 股價歷史連動（預設關閉） |

## ⚠️ 數據說明

本儀表板專注於**結構與關係**，不在資料庫內預存即時股價、本益比等市場數字（避免過時或不準）。即時行情、本益比、財報請點擊公司詳情面板下方的外部連結（Goodinfo、Yahoo 股市等）取得官方資料。

## 🗃️ 資料結構

### `data.js` 兩個主要資料結構：

```js
window.COMPANIES = [
  { code:"2330", name:"台積電", sector:"半導體", subSector:"晶圓代工",
    group:"獨立", rank:1, business:"...", products:[...], founded:1987 },
  // ...
];

window.LINKS = [
  // [from, to, type, label]
  ["2330","2454","supply","台積電代工聯發科"],
  ["2330","2303","industry","晶圓代工同業"],
  ["1301","1303","group","台塑四寶"],
  // ...
];
```

新增公司或連結直接編輯 `data.js` 即可，無需改動 `index.html`。
